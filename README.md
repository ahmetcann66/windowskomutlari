# Ağ ve Komut Satırı Araçları🛜

## `arp -a` Komutu
ARP (Adres Çözümleme Protokolü) önbelleğini gösterir.

```Bash
arp -a
```

![isim](Images1/arp%20-a.png " Komutu")

## `getmac` Komutu
Ağ adaptörlerinin MAC adreslerini gösterir.

```Bash
getmac
```

![isim](Images1/getmac.png " Komutu")

## `ipconfig/all` Komutu
Ağ adaptörlerinin tüm yapılandırma bilgilerini gösterir.

```Bash
ipconfig/all
```

![isim](Images1/ipconfig.png " Komutu")

## `nbtstat -n` Komutu
 NetBIOS ad tablosunu gösterir.

```Bash
nbtstat -n
```

![isim](Images1/nbtstat%20-n.png " Komutu")

## `net share` Komutu
Paylaşılan kaynakları gösterir veya yönetir.

```Bash
net share
```

![isim](Images1/net%20share.png " Komutu")

## `net user` Komutu
Kullanıcı hesaplarını listeler ve yönetir.

```Bash
net user
```

![isim](Images1/net%20user.png " Komutu")

## `netstat -an` Komutu
Ağ bağlantıları ve aktif portlar listelenir (IP ve port numarasıyla).
```Bash
netstat -an
```

![isim](Images1/netstat%20-an.png " Komutu")

## `ping -a` Komutu
Ping ile hedefin IP adresinin karşılık geldiği ana bilgisayar adı gösterilir.
```Bash
ping -a
```

![isim](Images1/ping%20-a.png " Komutu")

## `ping -f` Komutu

![isim](Images1/ping%20-f.png " Komutu")

## `ping -t` Komutu

```Bash

```

![isim](Images1/ping%20-t.png " Komutu")

## `ping` Komutu
Ping komutu, bir ağ cihazına veri paketleri göndererek o cihazın erişilebilir olup olmadığını ve yanıt süresini test eder.
```Bash

```

![isim](Images1/ping.png " Komutu")
